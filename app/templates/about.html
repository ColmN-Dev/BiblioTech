About BiblioTech

BiblioTech is a full-stack web application designed to help readers discover, organise, and review books in one place. Built using modern web technologies, the platform combines book discovery, personal library management, and user-generated reviews to create an enjoyable reading experience.

What is BiblioTech?

BiblioTech enables users to search for books using the Google Books API, explore detailed book information, save books to their personal library, and share ratings and reviews with other readers. The application focuses on providing a clean, responsive, and user-friendly experience while demonstrating the principles of modern full-stack software development.

Key Features

  • Search for books using the Google Books API.
  • Create a secure user account and authenticate using Flask-Login.
  • Save and manage a personal book collection.
  • Create, edit, and delete book reviews and ratings.
  • View detailed book information retrieved from external APIs.
  • Responsive interface designed for desktop and mobile devices.
  • Dynamic page rendering using the Jinja2 templating engine.

Technology Stack

  • Backend: Python, Flask, SQLAlchemy, PostgreSQL
  • Frontend: HTML5, CSS3, JavaScript
  • Templating: Jinja2
  • Authentication: Flask-Login & Flask Bcrypt Security
  • API Integration: Google Books API
  • Version Control: Git & GitHub

Project Goals

BiblioTech was developed as a practical full-stack software engineering project with the goal of applying real-world development practices. The project brings together frontend development, backend programming, database management, authentication, API integration, and dynamic web rendering into a single cohesive application.

The Meaning Behind the Name

The name BiblioTech combines "Biblio", derived from the Greek word biblios meaning "book", with "Tech", representing modern software development and technology. Together, the name reflects the project's purpose of connecting literature with contemporary web technologies.
